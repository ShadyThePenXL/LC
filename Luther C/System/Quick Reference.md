## Quick Reference — System Rules

This is the condensed ruleset. Read this before running the game.

---

### Stats

Seven attributes: Strength, Dexterity, Endurance, Intellect, Knowledge, Charm, Technique. Stats are numbers. No caps. Stat scaling is relative to body size — a stat point on a goblin is not the same as on a human. No HP. Wounds are physical consequences. Gold is silent — never ledgered. Imperial measurements only.

### Classes

Only sentient races (humans, elves, dwarves, fey, drow) have classes. Always written in brackets: [Hunter], [Mage], [Farmer]. Two base tiers: Common and Rare. Higher tiers (up to Legendary) only through evolution. Evolution is forced by the System when skills exceed the class framework. All skills are retained through evolution. At every 10-level threshold the System may offer a class change or evolution.

### Leveling

Experience from kills (System announces creature name and level). Leveling slows dramatically after every 10 levels. Skill points: 1 General + 1 Class every 5 levels. At class evolution: 3 options presented (1 new skill, 2 upgrades), pick 1.

### Skills

All skills have levels. Skills level through use. Soft cap every 5 skill levels — breaking through is a noticeable power jump. All skills use mana. In mana-dense environments (the Deeproot) ambient mana covers cost. Outside, sustained use drains the user. Detection skills pulse in bursts outside dense environments.

Skill categories: Class, Gifted (from gods), General (purchased from a massive specific list), Stolen (extracted via consumption).

Two or more overlapping skills can merge via System quest into something stronger than the sum. Divine skills require the god's release to merge.

Skill descriptions: descriptive language only ("increased," "improved," never percentages or flat numbers). No quips. No absolutes — always room to upgrade. No cross-referencing other skills in descriptions except as [Note: ...] addendums.

### Traits

Passive. Always on. Have levels or intensity stages. Sources: racial, monster/creature innate, or extracted. Can merge like skills via System quest. Biology is not always a trait — the System may not recognize a biological function.

Racial traits: Humans get none but level fastest. Elves get 2 (experience drops with age). Dwarves get 3 (same penalty).

### Magic

Common and everyday. Two paths: a casting class with built-in channeling, or buying cantrips from the general skill list (light, spark, minor push — nothing combat-winning). Any class can buy cantrips. Mages are extremely powerful in mana-dense environments.

### Creatures vs Monsters vs Classed

**Creatures:** Level and have traits. No stats per level. Act on biological impulse. Do not hunt or hold territory. Can still be lethal (venom, etc.).

**Monsters:** Level and have traits. Gain fixed stats per level by species. Hunt, hold ground, have instinct resembling decision-making. Stronger monsters have stronger traits. Some develop skills (Elites) — pattern unknown, more common in the Deeproot.

**Classed:** Sentient races with System-granted classes.

Categories blur — farmland wolves become creature-dogs over generations. Same species, different classification based on nature.

### Predator's Claim / Mandate

Claim: fires on full consumption. One trait attempt per creature. Can be directed. Usually yields partial/reduced version. Repeated claims of same type feed existing trait. Failed claims yield partial stat growth. Must be System-recognized.

Mandate: scans for one skill AND one trait simultaneously. Directed claims carry no additional cost. If no skill exists, skill channel sits empty. Enchantments and non-living mana structures can be studied through repeated consumption and eventually reproduced.

### Mass-Consumption Extraction

When sources are too small for individual extraction, a progress bar appears. Accumulates across many kills. Royal guard-tier sources give dramatically more progress. Trait acquired at 100%.

### Mana Well

Fills from ambient mana. Deeproot at Level 7: ~8 seconds empty to full. Outside Deeproot: dramatically slower.

### Hide Status

Can hide/blank information. Cannot lie or replace with false data. The System refuses false input.

### The Crowned Feast

Ten distinct colony species queens required. Same species beyond the first does not count. Classed queens improve the reward. Exclusively classed queens improve it further. Cannot be refused, abandoned, or forgotten.

### Skill Merging (Divine Exception)

Divine skills (granted by gods) cannot be merged through the System alone. The granting god must release the skill. The god sets the price.
