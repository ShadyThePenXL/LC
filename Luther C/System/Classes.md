## Classes

Only truly sentient races have classes — humans, elves, dwarves, fey, drow. Monsters, goblins, and creatures do not have classes. Their skills and traits are race-based or biologically innate.

Classes are always written in brackets: [Hunter], [Devourer], [Mage], [Farmer].

### Tiers

- **Common** — The base tier. Most people have a Common class.
- **Rare** — The next tier up. Most Common classes can evolve into a Rare class.

Nothing higher than Rare exists at baseline. Higher tiers (up to Legendary) only appear through class evolution.

### Class Evolution

Class evolution can be forced by the System when the bearer's skills exceed the class framework's structural tolerance. The class can no longer contain what the bearer is doing with it.

When evolution triggers, the System presents options ranging from Rare to Legendary depending on circumstances — the bearer's skill composition, achievements, and current trajectory.

All current skills are retained through evolution regardless of which option is selected.

### Class Change

At or around each 10-level threshold, the System may offer a class change, class evolution, or other structural options depending on the individual's skills, achievements, and circumstances.

### Stat Growth from Classes

Each class has a fixed stat spread — a set number of attribute points gained per level. The spread is determined by the class and does not change. Different classes weight different attributes.
